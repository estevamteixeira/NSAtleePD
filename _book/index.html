Welcome

This is the online home of the Nova Scotia Atlee Perinatal Database, a report meant to provide a quick reference to the sentinel indicators of perinatal health and care among Nova Scotia residents.

Note: Previous editions of this report can be found on the RCP web site.

Inspired by the Free and Open Source Software movement (FOSS), the code and prose underlying this report are open, ensuring that the content is reproducible, transparent, and accessible Hosting the source code on GitHub allows anyone to interact with the project by opening issues or contributing new content and typo fixes for the benefit of everyone.

The online version of the report is hosted at [rcp.nshealth.ca/publications]http://rcp.nshealth.ca/publications/nsapd-report-indicators-2010-2019).
